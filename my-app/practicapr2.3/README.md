# INSTRUCCIONES DE USO #

Este proyecto esta hecho con la herramienta de software Maven:

*  Compilar: 
mvn compile
*  Testear:
mvn test
*  Generar javadoc:
mvn javadoc:javadoc
*  Generar jar:
mvn package
